# Playstore - A Modern Fullstack playstore app website version
Built with the Next.js App, Node.js, Router, TypeScript, CSS
## Features
* Infinite scrolling for dynamically loading posts
* Optimistic updates for a great user experience
* Modern data fetching using React-Query
* Full Search functionality
* ... and much more
## Getting started
To get started with this project Download the code in your local mechine <br>
run these commands <br>
Enter root directory: <br>
```
cd Playstore_Web_1-main
```
### from Root Directory:
For Back end: <br>
```
cd backend
```
For dependencies to install: <br>
```
npm install
```
For front end: <br>
```
cd frontend
```
For dependencies to install: <br>
```
npm install
```
From here your ready to run the code in your local machine
### Steps to run:
Run Backend: <br>
from root Directory
```
cd backend
npm run server
```
Run Frontend: <br>
from root Directory
```
cd frontend
npm run dev
```
Backend runs on Localhost:8080 , open http://localhost:8080/ <br>
Frontend runs on Localhost:5173 , open http://localhost:5173/
